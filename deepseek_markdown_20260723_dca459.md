# Telegram Message Copier

A fully automated Telegram message copier that runs 24/7 on Railway.

## Features
- 🔄 Real-time message copying
- 🔍 Keyword filtering (include/exclude)
- 🔗 Text replacement
- 📷 Media support (images, videos, files)
- 🚀 Runs 24/7 on Railway

## Setup

### 1. Get Telegram Credentials
- Go to my.telegram.org/apps
- Create a new application
- Get your API_ID and API_HASH

### 2. Generate String Session
```bash
pip install telethon
python generate_session.py
